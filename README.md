'NFT' to 'silly jpeg'
=============

![](screen.png)

Chrome extension that replaces all occurrences of 'NFT' with 'silly jpeg'.

Forked from PanicSteve's [cloud-to-butt](https://github.com/panicsteve/cloud-to-butt/)
